🏠  1. OVERVIEW & GETTING STARTED▸

Eyethu Pro Radio Studio is a fully self-contained, single-file Progressive Web App (PWA) and native Android APK designed for professional radio automation. It runs entirely in your browser with no server, no installation and no internet required — or as a native app via the included APK.

Works on Android phones, tablets, desktop and laptop computers. The native Android APK streams directly to Icecast/Shoutcast with no relay server required.

STUDIODual-deck playback, crossfader, queue, cartwall, mic, channel console
LIBRARYImport individual files or load an entire music folder
SCHEDULEAutomated show scheduling with intro/outro carts and ad breaks
SETTINGSDSP chain display, fader defaults, presets, device routing, live streaming
💡 First time? Go to Library → tap 📂 IMPORT FILES to pick songs or 📁 LOAD FOLDER to load a music folder. Then go to Studio, enable AUTO-DJ: ON and press ▶ on Deck A to start broadcasting.
🎛  2. STUDIO TAB▸

The Studio tab is your main broadcast surface. It contains the dual decks, crossfader, queue, cartwall, microphone panel with ON AIR buttons, and the channel console.

▸ DECK A & DECK B
📂Load a single audio file directly onto this deck
▶ / ⏸Play or pause the loaded track
■Stop and return to beginning
CUESet a cue point at the current playback position
↩Jump back to the saved cue point
⏏Eject — unloads the track and frees memory
VOLDeck volume fader — default 81%
Progress barTap anywhere to seek to that position
▸ CROSSFADER

Blends between Deck A (left) and Deck B (right) using constant-power curves. The ⟺ AUTO XFADE button smoothly slides the crossfader to the opposite side over 3 seconds.

▸ QUEUE & AUTO-DJ

The queue holds tracks in playback order. Tap any Library track to add it. Auto-DJ automatically loads the next queued track onto the opposite deck when a song ends, creating a seamless A/B loop.

AUTO-DJ: ON/OFFToggle automatic next-track loading and playback
🔀Shuffle the queue into a random order
✕ CLEARRemove all tracks from the queue
▸ CARTWALL

8 instant-play cart buttons for jingles, station IDs, ads and sound effects. Each cart ducks the music by −12 dB when playing and restores it smoothly when done.

Tap oncePlay the cart — music ducks −12 dB
Tap againStop the cart immediately — music restores
Long-pressLoad an audio file onto that cart slot
F1–F8 keysDesktop: trigger carts 1–8 from keyboard (play/stop toggle)
▸ MICROPHONE & ON AIR
🎤 MIC ONRequest mic access and activate mic pre-processing chain
🔄 RECONNECTReconnect mic if the browser disconnected it
VOLMic pre-gain — default 90%
🔴 ON AIRMutes studio monitors, opens mic live, activates −12 dB auto-ducking
⚪ OFF AIRRestores studio monitors, gates mic, releases ducking over 800ms
▸ CHANNEL CONSOLE

5-channel mixer (Deck A, Deck B, Mic, Cart, Master) with vertical faders and live VU meters. All faders are bi-directionally linked — moving a console fader moves the corresponding deck/mic slider in real time.

▸ KEYBOARD SHORTCUTS (Desktop)
F1 – F8Cart 1–8 play/stop toggle
Space / PPlay / pause the active deck
💡 F-key badges (F1–F8) appear on cart buttons on desktop so you always know which key to press. They are hidden on touch devices.

📅  4. SHOW SCHEDULER▸

The Scheduler pre-programs shows that start and stop automatically at set times. Each show has its own playlist, ad break settings and intro/outro carts.

⚙️  5. SETTINGS▸
▸ Mic Pre-Processing Chain

Applied to the microphone only, before it enters the mix bus. Does not affect music.

HIGHPASS 80 HzRemoves handling noise, breath and rumble
NOISE GATE −45 dBSilences mic between speech — 20:1 ratio
SAFETY LIMITER −6 dBHard ceiling — prevents mic from overloading the mix bus
▸ Master DSP Chain

Applied to the full mix bus — all sources (Decks, Cart, Mic) pass through this single chain. The multiband compressor was removed because Web Audio's automatic makeup gain (+17 dB) was the root cause of distortion.

PRE-TRIM ×0.40Brings mix bus to safe −8 dBFS before any processing
HIGHPASS 35 HzRemoves sub-bass rumble from the full mix
AGC LEVELERThreshold −18 dB · Ratio 1.2:1 · Attack 500ms · Release 4s — transparent, only +1.8 dB makeup
POST-AGC TRIM ×0.82Cancels AGC's +1.8 dB makeup — keeps level flat
SOFT CLIPPER 0.20Gentle algebraic saturation — rounds peaks before limiting
LOOKAHEAD LIM −1 dB5ms lookahead — transparent peak control
BRICKWALL −0.5 dBAbsolute safety ceiling — zero digital clipping guaranteed

📊  6. VU METERS & LEVELS▸

All meters are 20-segment LED-style VU meters displaying RMS level with a peak-hold indicator.

GREEN (0–65%)Normal operating level — safe zone
YELLOW (65–85%)Approaching hot — reduce level if sustained
RED (85–100%)High level — brickwall limiter will engage
PEAK DOTWhite-outline segment that holds briefly at peak, then decays
💡 Aim for green-to-low-yellow during normal playback. The brickwall limiter prevents clipping but sustained red causes audible limiting artefacts.
